#  Lambda MongoDB Backup
## Summary
AWS Lambda functions for MongoDB database backups
## Environment variables
| Key               | comment                        |
|-------------------|--------------------------------|
| CONNECTION_STRING | your MongoDB connection string |
| S3_BUCKET         | S3 bucket name                 |
| S3_REGION         | S3 bucket Region               |
